Описание проекта
Проект «Сложно сосредоточиться» — это веб-сайт, который создан с адаптивность под разные устройства и темы экрана. Сам сайт рассказывает почему нам так трудно сконцентрироваться.

Технические требования
HTML
CSS
JavaScript
Инструкции по запуску
Скачайте файлы проекта или клонируйте репозиторий.
Откройте файл index.html в браузере.
Структура проекта
index.html — основной HTML-файл.
css/ — папка с CSS-файлами.
js/ — папка с JavaScript-файлами.
